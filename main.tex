\chapter{Практична частина}
Тут ми наведемо приклади використання лістингу коду.
\begin{lstlisting}[language=Python, caption=Простий приклад коду Python, label={lst:python_example}]
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
\end{lstlisting}
Як показано в лістингу \ref{lst:python_example}, ця функція рекурсивно обчислює факторіал.

% приклад алгоритму
\begin{algorithm}
\caption{Обчислення факторіалу}
\label{alg:factorial} % Унікальний ідентифікатор алгоритму
\begin{algorithmic}[1]
\Function{Factorial}{$n$}
    \If{$n = 0$}
        \State \Return 1
    \Else
        \State \Return $n \cdot$ \Call{Factorial}{$n-1$}
    \EndIf
\EndFunction
\end{algorithmic}
\end{algorithm}
% --------------------------------------------------------------------------------------
Як показано в алгоритмі \ref{alg:factorial}, рекурсивний виклик використовується для обчислення факторіалу.
