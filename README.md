# 📘 LaTeX‑шаблон для курсової / дипломної роботи

Цей репозиторій містить повноцінний LaTeX‑шаблон для оформлення курсових та дипломних робіт українською мовою.  
Шаблон побудований модульно, підтримує XeLaTeX, містить набір макросів і приклади їх використання.

---

## 🚀 Швидкий старт

### 1) Використання онлайн (Overleaf)

1. Імпортуйте репозиторій у Overleaf або завантажте файли вручну.
2. У налаштуваннях проекту встановіть компілятор **XeLaTeX**.
3. Запустіть компіляцію — Overleaf автоматично обробить усі `.tex` файли.

### 2) Локальна робота (Windows / macOS / Linux)

1. Встановіть повну TeX‑систему: **TeX Live** або **MiKTeX**.
2. Переконайтеся, що у системі доступні шрифти, які використовуються у преамбулі.
3. Рекомендується використовувати `latexmk` для автоматичної компіляції:

```bash
latexmk -xelatex -pvc -f main.tex -outdir=build
```

## 📂 Структура проєкту

```
├── main.tex                # Головний файл документа
├── preamble/               # Пакети, стилі, макроси
│   ├── packages.tex
│   ├── macros.tex
│   ├── styles.tex
│   ├── layout.tex
│   └── headings.tex
├── sections/               # Розділи роботи
│   ├── 01_intro.tex
│   ├── 02_section.tex
│   └── ...
├── assets/                 # Зображення та додаткові файли
│   └── img.png
└── build/                  # Вихідні та проміжні файли (включно з main.pdf)
```

## 🧩 Користувацькі макроси

### 1) `\maketitlepage` — титульна сторінка

Формує титульну сторінку з усіма необхідними полями (інститут, факультет, тема, студент, керівник).  
Приклад використання наведено у `main.tex`.

---

### 2) Розділи та підрозділи

Шаблон містить обгортки над стандартними командами LaTeX для зручності та контролю відступів.

#### `\customSection[мітка]{Назва}{відступ}`

```latex
\customSection[sec:architecture]{Проєктування та архітектура системи}{0em}
```

```latex
\customSubSection[sec:architecture_two_level]{Секція другого рівня}{1em}
```

```latex
\customSubSubSection[sec:analysis_third_level]{Секція третього рівня}{1em}
```

### 2) Вставка рисунків

```latex
\setfigure[fig:architecture_example]{0.8\textwidth}{assets/img}{Ілюстрація}
```

```latex
\setfigure[fig:architecture_example]{0.8\textwidth}{assets/img}{Ілюстрація}
```

Приклад у тексті:

```latex
На \textbf{рис.~\ref{fig:architecture_example}} наведено приклад.

\setfigure[fig:architecture_example]{0.8\textwidth}{assets/img}{Ілюстрація щодо проєктування та архітектури системи}
```

## 🏗️ Порядок роботи з документом
1. **Редагуйте текст** у файлах `sections/*.tex` — кожен розділ зберігається окремо.
2. **Глобальні стилі, макроси та налаштування** знаходяться у каталозі `preamble/`.
3. **Зображення та додаткові матеріали** додавайте в `assets/` і вставляйте через макрос `\setfigure`.
4. Для автоматичної перекомпіляції використовуйте команду:

```bash
latexmk -xelatex -pvc -outdir=build main.tex
```
