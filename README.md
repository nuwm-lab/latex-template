# Шаблон LaTeX для курсових робіт / проєктів

Цей репозиторій містить простий шаблон `main.tex` для написання курсових робіт та проєктів студентів спеціальності F2 — Інженерія програмного забезпечення (бакалавр і магістр) Національного університету водного господарства та природокористування.

## Для чого це
- Забезпечити уніфікований формат оформлення робіт: нумерацію розділів, заголовків, підписів до рисунків і таблиць.
- Встановити стандартні шрифти, відступи, інтервали та правила для структурування документу.
- Надати готовий файл `main.tex`, який можна використати як стартовий шаблон.

## Як користуватись
1. Скопіюйте (або завантажте) `main.tex` у свій проєкт.
2. Найпростіший спосіб працювати — завантажити `main.tex` на https://www.overleaf.com/ і редагувати онлайн. Overleaf автоматично компілює документ і показує результат у реальному часі, що дуже зручно при підготовці тексту та перевірці форматування.
3. Заповнюйте заголовок, авторські дані, зміст, розділи, рисунки і бібліографію відповідно вашої роботи / проєкту.

## Що задає цей шаблон
- Правила нумерації розділів, підрозділів, рисунків і таблиць.
- Стандартні шрифти і розміри для основного тексту та заголовків.
- Можливі налаштування для полів сторінки, міжрядкових інтервалів та відступів.
- Місця для підключення зображень і посилань на бібліографію.

## Ліцензія
Ви можете використовувати цей шаблон в освітніх цілях. Якщо хочете — додайте свою ліцензію або вкажіть авторів у власному репозиторії.
